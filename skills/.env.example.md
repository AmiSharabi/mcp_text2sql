Create `.env.example` containing all configuration keys required for the POC (no real secrets). Include:
- MCP_HOST=0.0.0.0
- MCP_PORT=8000
- DB_HOST=localhost
- DB_PORT=1433
- DB_NAME=YourDb
- DB_USER=readonly_user
- DB_PASSWORD=replace_me
- DB_DRIVER=ODBC Driver 17 for SQL Server
- DB_ENCRYPT=yes
- DB_TRUST_SERVER_CERTIFICATE=yes
- SQL_TIMEOUT_SECONDS=10
- SQL_MAX_ROWS=200
- SCHEMA_CACHE_SECONDS=300
- LOG_PATH=logs/events.jsonl
Optionally include OPENAI_API_KEY placeholder. Do not commit `.env` with real secrets.
